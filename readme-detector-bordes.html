🔍 Detector de Bordes - Sobel

✅ Detección de bordes con Sobel
✅ Conversión a escala de grises

} private Bitmap AplicarSobel(Bitmap imagenOriginal) { Bitmap imagenGris = ConvertirAGrises(imagenOriginal); int ancho = imagenGris.Width; int alto = imagenGris.Height; Bitmap resultado = new Bitmap(ancho, alto); int[,] sobelX = { { -1, 0, 1 }, { -2, 0, 2 }, { -1, 0, 1 } }; int[,] sobelY = { { -1, -2, -1 }, { 0, 0, 0 }, { 1, 2, 1 } }; for (int y = 1; y < alto - 1; y++) { for (int x = 1; x < ancho - 1; x++) { int gx = 0, gy = 0; for (int ky = -1; ky <= 1; ky++) { for (int kx = -1; kx <= 1; kx++) { Color pixel = imagenGris.GetPixel(x + kx, y + ky); int valor = pixel.R; gx += sobelX[ky + 1, kx + 1] * valor; gy += sobelY[ky + 1, kx + 1] * valor; } } int magnitud = (int)Math.Sqrt(gx * gx + gy * gy); magnitud = Math.Min(255, magnitud); Color color = Color.FromArgb(magnitud, magnitud, magnitud); resultado.SetPixel(x, y, color); } } return resultado; } private Bitmap ConvertirAGrises(Bitmap original) { Bitmap gris = new Bitmap(original.Width, original.Height); for (int y = 0; y < original.Height; y++) { for (int x = 0; x < original.Width; x++) { Color pixel = original.GetPixel(x, y); int promedio = (int)(pixel.R * 0.3 + pixel.G * 0.59 + pixel.B * 0.11); Color grisColor = Color.FromArgb(promedio, promedio, promedio); gris.SetPixel(x, y, grisColor); } } return gris; } private void GuardarImagen() { if (pbProcesada.Image == null) { MessageBox.Show("Procesa una imagen primero"); 📖 Cómo Funciona el Algoritmo de Sobel

Paso 1: Conversión a Escala de Grises

Gris = (R × 0.3) + (G × 0.59) + (B × 0.11)

Esta fórmula pondera cada canal RGB para mantener la luminosidad correcta.

Paso 2: Máscaras de Sobel

Se aplican dos matrices de convolución 3×3:

Gx (Horizontal) Gy (Vertical) [-1, 0, 1] [-1, -2, -1] [-2, 0, 2] [ 0, 0, 0] [-1, 0, 1] [ 1, 2, 1]

Paso 3: Cálculo del Gradiente

Magnitud = √(Gx² + Gy²)

💡 Consejos para Mejores Resultados

  • Alto contraste: Las imágenes con buen contraste producen bordes más claros
  • Imágenes nítidas: Evita desenfoque, el algoritmo funciona mejor con bordes definidos
  • Resolución: Procesa imágenes de hasta 4000×4000 píxeles
  • Formato PNG: Mejor para archivos de origen antes de procesar

🐛 Solución de Problemas

❌ Error: "InitializeComponent no existe"
✅ Solución: Usa el método CrearInterfaz() en lugar de InitializeComponent()
❌ Error: "Program ya contiene una definición para Main"
✅ Solución: Elimina archivos duplicados (Form1.cs, Program.cs extra)
❌ Error: "No se puede cargar la imagen"
✅ Solución: Verifica que el archivo no esté corrupto y está en formato válido
❌ La imagen se ve en blanco
✅ Solución: La imagen debe tener contraste para detectar bordes visibles
